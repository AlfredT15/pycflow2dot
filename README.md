pycflow2dot
===========

Description
-----------
PyCflow2dot is a Python port of the Perl script cflow2dot.
Tested with Python 3.2 (NetworkX not yet available in 3.3.).

Uses cflow and dot to draw a call graph.
Available output formats: .DOT, .SVG, .PDF, .PNG.
